# ✨ Vanilla JavaScript Apps

Welcome to the **Vanilla JavaScript Apps** repository! This project serves as a comprehensive collection of lightweight, interactive web applications built entirely using pure, native JavaScript, HTML5, and CSS3. 

No frameworks (like React, Vue, or Angular) or heavy libraries are used here—just clean, vanilla web fundamentals.

---

## 📂 Repository Structure

This repository contains the following mini-projects:

| Folder Name | Project Description | Key Features |
| :--- | :--- | :--- |
| 🎨 **`Color Flipper`** | A dynamic color generator tool. | Generates random hex/simple colors and updates the background in real-time. |
| 🎲 **`Guess Number Game`** | A classic number guessing game. | Random number generation, score tracking, user input validation, and high score records. |
| ⏱️ **`Js Clock`** | A real-time digital or analog web clock. | Utilizes JavaScript's `Date` object and intervals to accurately display the time. |
| 🎵 **`Music Player App`**| A lightweight web audio player. | Audio controls (Play, Pause, Skip, Vol), progress bar tracking, and playlist handling. |
| 📝 **`Quiz`** | An interactive trivia/knowledge quiz app. | Multiple-choice questions, score calculation, progress indicators, and a final results screen. |
| 🛒 **`Shopping Cart`** | A mock e-commerce item management cart. | Add/remove items, update quantities, and calculate real-time totals with dynamic UI updates. |
| 💸 **`Tip calculator`** | A quick utility for calculating service tips. | Calculates individual or split tip amounts based on bill total and service ratings. |
| ✅ **`ToDoList`** | A simple task management application. | Create, edit, mark as complete, or delete tasks. Includes local storage persistence features. |
| 🧮 **`Calculator`** | A fully functional digital calculator. | Standard arithmetic operations (addition, subtraction, multiplication, division) and display parsing. |

---

## 🛠️ Built With

* **HTML5:** Semantic web structure
* **CSS3:** Custom layouts, responsiveness, and styling
* **Vanilla JavaScript (ES6+):** DOM manipulation, event listeners, and application logic

---

## 🚀 Getting Started

Since these applications rely exclusively on client-side vanilla code, you do not need to install any servers, node modules, or compile steps to run them.

### 🛠️ Prerequisites
All you need is a modern web browser (e.g., Google Chrome, Mozilla Firefox, Microsoft Edge, Safari).
